# 提交信息模板

## 格式
```
<类型>: <简短描述>

<详细描述>

<可选的脚注>
```

## 类型
- `feat`: 新功能
- `fix`: 修复 bug
- `docs`: 文档更新
- `style`: 代码格式调整
- `refactor`: 代码重构
- `test`: 测试相关
- `chore`: 构建过程或辅助工具变动
- `perf`: 性能优化
- `ci`: CI 配置

## 示例
```
feat: 添加 Telegram 私聊总结系统

- 实现数据收集和归档功能
- 添加 AI 总结生成器
- 配置定时任务调度
- 添加部署验证脚本

Closes #123
```

```
- 核心功能: 私聊会话数据收集和归档

配置说明:

```
